# EquirectCameraTest
Equirectangular Cameraテスト用Unityプロジェクト

Equirectangularカメラのみを自分のプロジェクトに追加する際は以下のようにしてください。

1. Releaseにある[Unity Package](https://github.com/ktaka/EquirectCameraTest/releases/download/EquirectCamera_v1.0/URP_EquirectangularCamera_unity6.unitypackage)をダウンロードする
2. ダウンロードしたUnity Packageをプロジェクトにインポートする
3. Assets/URPEquirectangularCameraフォルダにあるEquirectCameraプレハブをシーンに入れる
4. 元々あったMain CameraはOFF、または削除する
